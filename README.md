## Sobre
O projeto contém tópicos sobre os fundamentos do banco de dados relacional PostgreSQL.

## Tópicos disponíveis
* Introdução ao curso
* Noções sobre banco de dados
* Instalando o PostgreSQL
* Acessando o PostgreSQL
* Criando um banco de dados
* Alterando tabelas
* Manipulando dados
* Filtrando e agrupando dados
* Relacionamento 1:1
* Relacionamento 1:N
* Relacionamento N:N
* pgAdmin
* Stored procedures
* Transações
* Subselects
* Functions
* Triggers
* Segurança
* Backup e Restore
